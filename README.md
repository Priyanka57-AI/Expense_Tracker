# 💰 Smart Expense Tracker AI

An intelligent expense tracking web app built using **Streamlit**, enhanced with **AI-powered insights** and **financial data integration**.

---

## 🚀 Features

* ➕ Add and manage daily expenses
* 📋 View all recorded expenses in a structured table
* 📊 Visual analytics with category-wise spending
* 🤖 AI-generated insights on spending behavior
* 📈 Live stock market data integration
* 🗄️ Database storage for persistence

---

## 🛠️ Tech Stack

* **Frontend & App Framework:** Streamlit
* **Backend Logic:** Python
* **Database:** SQLite
* **AI Integration:** Transformers / NLP-based insights
* **Finance API:** Stock data integration (e.g., Yahoo Finance)

---

## 📂 Project Structure

```
Expense_Tracker/
│── app.py              # Main Streamlit app
│── expense.py          # Expense class
│── utils.py            # Add/View expense functions
│── analytics.py        # Data analysis logic
│── ai_insights.py      # AI-generated insights
│── finance_api.py      # Stock data fetching
│── db.py               # Database setup
│── requirements.txt    # Dependencies
```

---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/Priyanka57-AI/Expense_Tracker.git
cd Expense_Tracker
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```

---

## 🧠 How It Works

* Users can **add expenses** with title, amount, category, and date
* Data is stored in a **SQLite database**
* Analytics module processes data for **visual insights**
* AI module generates **smart suggestions and spending analysis**
* Finance module fetches **real-time stock data**
